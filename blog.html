What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface that represents web documents (like HTML or XML) as a tree of objects. It allows scripts, mainly JavaScript, to dynamically access and manipulate the document's structure, content, and styles.

How do you select an element from the DOM?

You can select elements from the DOM using several methods in JavaScript. The getElementById('id') method allows you to select an element by its unique ID. For selecting all elements with a specific class name, you can use getElementsByClassName('className'), while getElementsByTagName('tagName') selects all elements with a given tag name. For more versatile selections, querySelector('selector') retrieves the first matching element based on a CSS selector, and querySelectorAll('selector') returns all matching elements. These methods provide powerful ways to access and manipulate elements within a web document.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a method where you attach one event listener to a parent element instead of adding listeners to each child element. When a child element is clicked or interacted with, the event goes up to the parent, which can handle it. This is helpful because it saves memory, works with new child elements automatically, and makes your code simpler. Overall, event delegation improves performance and makes managing events easier.

Ho do you manipualte an element's attributes and styles usingthe dom?

To manipulate an element's attributes and styles using the DOM, you can use simple JavaScript methods. To get an attribute value, use getAttribute(), and to change or add an attribute, use setAttribute(). If you want to remove an attribute, removeAttribute() does the job. For styles, you can change things like background color or font size directly using the style property. You can also manage CSS classes: use classList.add() to add a class, classList.remove() to remove one, and classList.toggle() to switch a class on or off. These methods make it easy to change how elements look and behave on your web page.
